# Quick Task 260604-l64 Summary: Add batching for paragraphs translation

## Result

Implemented paragraph batch translation with structured JSON requests/responses.

## Changes

- Added deterministic translation batches sized toward 60% of an estimated context budget.
- Added context selection:
  - chapter title and heading context at chapter/section start;
  - up to 3 previous same-chapter paragraphs, including translations when present;
  - no cross-chapter context leakage.
- Changed prompt payload to JSON with source/target languages, context entries, and item `{id,text}` records.
- Forced OpenAI structured output with `response_format={"type":"json_schema", ...}`.
- Parsed batch response by paragraph ID, preserving valid translations and marking missing/malformed outputs as `[TRANSLATION FAILED]`.
- Kept output document JSON shape, destination filename logic, CLI options, and EPUB assembly path unchanged.
