    `class JobStore`:
    - `__init__(self, base: Path = RUNS_BASE) -> None` — assigns `self.base = base`, calls `self.base.mkdir(parents=True, exist_ok=True)`
    - `create_run(self, meta: JobMeta) -> str` — generates `run_id = uuid.uuid4().hex[:12]`, creates `run_dir / "src"` and `run_dir / "dst"` via `mkdir(parents=True)`, calls `self._write_meta(run_dir, meta)`, returns `run_id`
    - `_write_meta(self, run_dir: Path, meta: JobMeta) -> None` — writes to `run_dir / "meta.json.tmp"` with `json.dumps({"model": meta.model, "params": meta.params}, indent=2)` encoded as UTF-8, then calls `os.replace(tmp_path, run_dir / "meta.json")` — NEVER write directly to `meta.json`
    - `read_meta(self, run_id: str) -> JobMeta` — reads `self.base / run_id / "meta.json"` as UTF-8 JSON, returns `JobMeta(model=data["model"], params=data.get("params", {}))`
    - `update_meta(self, run_id: str, meta: JobMeta) -> None` — calls `self._write_meta(self.run_dir(run_id), meta)` (same atomic write)
    - `list_runs(self) -> list[str]` — returns `sorted(p.name for p in self.base.iterdir() if p.is_dir())`
    - `run_dir(self, run_id: str) -> Path` — returns `self.base / run_id`
    - `src_dir(self, run_id: str) -> Path` — returns `self.run_dir(run_id) / "src"`
    - `dst_dir(self, run_id: str) -> Path` — returns `self.run_dir(run_id) / "dst"`

    Imports: `from __future__ import annotations`, `import json`, `import os`, `import uuid`, `from pathlib import Path`, `from book_translator.models.job import JobMeta`.

    Do NOT store `lang_from`, `lang_to`, or `book_name` in `meta.json` — these are derived from file names (JOB-02 / JOB-03).
    Do NOT use `path.write_text(json.dumps(...))` directly for meta.json — always go through `_write_meta`.
