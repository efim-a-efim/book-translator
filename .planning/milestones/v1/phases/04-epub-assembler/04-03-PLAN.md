**Implement `assemble(job_dir: Path, target_lang: str) -> Path`:**

1. **Discover translated JSON:**
   ```python
   json_files = list((job_dir / "dst").glob("*.json"))
   if len(json_files) != 1:
       raise ValueError(
           f"Expected exactly 1 JSON in {job_dir / 'dst'}, found {len(json_files)}: {json_files}"
       )
   json_path = json_files[0]
   ```

2. **Load BookDocument:**
   ```python
   doc = BookDocument.from_json(json_path.read_text(encoding="utf-8"))
   ```

3. **Derive book_name and output path (D-14):**
   ```python
   book_name = json_path.stem.rsplit(".", 1)[0]   # e.g. "my_book.en" -> "my_book"
   epub_path = job_dir / "dst" / f"{book_name}.{target_lang}.epub"
   ```

4. **Build EpubBook:**
   ```python
   book = EpubBuilder().build(doc, target_lang, book_id=str(job_dir.name))
   ```

5. **Atomic write (D-06, matches engine.py pattern):**
   ```python
   tmp_path = epub_path.with_suffix(".epub.tmp")
   epub.write_epub(str(tmp_path), book, {})
   os.replace(tmp_path, epub_path)
   ```
   Use `str(tmp_path)` because ebooklib expects a string path, not `Path`.

6. **Return:** `return epub_path`

**Final module imports needed:**
```python
from __future__ import annotations
import os
from pathlib import Path
from ebooklib import epub
from book_translator.models.document import BookDocument
from book_translator.assembler.builder import EpubBuilder
```

**`__all__`:** `["assemble"]`

The function must NOT silently ignore the case where `dst/` contains zero or multiple JSON files — the `ValueError` is intentional (fail fast with a clear message).

## Threat Model

| Threat | Category | Component | Disposition | Mitigation |
|--------|----------|-----------|-------------|------------|
| T-04-07 | Tampering | `assemble()` — output path derived from `json_path.stem`; if JSON filename contains `..` or path separators, could write outside dst/ | mitigate | `epub_path` is always constructed as `job_dir / "dst" / f"{book_name}.{target_lang}.epub"`. `book_name` is derived from a filename stem (no directory separators); `target_lang` is a language code from CLI (Phase 5 validates format). Path is fully resolved under `job_dir/dst/`. |
| T-04-08 | DoS | `assemble()` — writing large EPUB to disk; `os.replace` is atomic | accept | File system atomicity ensures no corrupt partial files; disk space exhaustion is an OS-level concern |
| T-04-09 | Information Disclosure | Leaked `.epub.tmp` file on unexpected crash mid-write | accept | Python exception during `epub.write_epub()` may leave `.tmp`; caller (Phase 5 CLI) should clean up on error. Acceptable for v1. |
