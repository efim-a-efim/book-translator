**`_find_source_json(job_dir: Path) -> Path`:**
Implements D-09 source file discovery. Globs `job_dir / "src"` for `*.json` files.
- If exactly 1 match: return it.
- If 0 or 2+ matches: raise `TranslationError(f"Expected exactly 1 JSON in {job_dir / 'src'}, found {len(matches)}: {matches}")`.

```python
def _find_source_json(job_dir: Path) -> Path:
    matches = list((job_dir / "src").glob("*.json"))
    if len(matches) != 1:
        raise TranslationError(
            f"Expected exactly 1 JSON in {job_dir / 'src'}, found {len(matches)}: {matches}"
        )
    return matches[0]
```

**`_dst_path(src_path: Path, job_dir: Path, target_lang: str) -> Path`:**
Implements D-09 destination filename convention. The source file name convention is `<name>.<source_lang>.json` (e.g., `war-and-peace.ru.json`). Strip the source language suffix to get the base name:
- `stem = src_path.stem` → `"war-and-peace.ru"` (Path.stem strips the `.json` extension)
- `name = stem.rsplit(".", 1)[0]` → `"war-and-peace"` (strips the source lang suffix)
- Return `job_dir / "dst" / f"{name}.{target_lang}.json"`

```python
def _dst_path(src_path: Path, job_dir: Path, target_lang: str) -> Path:
    stem = src_path.stem
    name = stem.rsplit(".", 1)[0]
    return job_dir / "dst" / f"{name}.{target_lang}.json"
```

**`translate(...)` — the full public entry point (D-10):**

Replace the stub with the full implementation:

```python
async def translate(
    job_dir: Path,
    model: str,
    api_key: str,
    base_url: str | None,
    source_lang: str,
    target_lang: str,
    context_window: int = 3,
    concurrency: int = 5,
    max_retries: int = 5,
) -> None:
```

Body:
1. `src_file = _find_source_json(job_dir)` — find the BookDocument JSON in src/
2. `doc = BookDocument.from_json(src_file.read_text(encoding="utf-8"))` — deserialize
3. `flat = [p for ch in doc.chapters for p in ch.paragraphs]` — flatten ALL paragraphs across ALL chapters; cross-chapter context is automatic because the flat list has no chapter boundaries
4. `semaphore = asyncio.Semaphore(concurrency)` — create ONCE, shared across all coroutines
5. `async with create_client(api_key, base_url) as client:` — single client per job, uses context manager for connection pool teardown; `create_client` is already imported from `client.py`
6. Define inner coroutine `translate_one(idx: int, para: Paragraph) -> None`:
   - **Skip guard (D-11, D-12):** `if para.kind in ("image", "table") or not para.text: return` — image/table paragraphs and empty-text paragraphs are skipped; `translation` stays `None`
   - Build context: `before, after = build_context_window(flat, idx, context_window)`
   - Build messages: `sys_prompt = build_system_prompt(source_lang, target_lang)` then `user_msg = build_user_message(para, before, after)` then `messages = [{"role": "system", "content": sys_prompt}, {"role": "user", "content": user_msg}]`
   - Call: `para.translation = await translate_paragraph(client, model, messages, max_retries, semaphore)` — `translate_paragraph` returns a string (translated text or "[TRANSLATION FAILED]"), never raises for retryable exhaustion
   - For non-retryable exceptions re-raised by `translate_paragraph` (e.g., 401): wrap in `try/except Exception as exc:` → `logger.warning("Paragraph %s failed (non-retryable): %s", para.id, exc)` → `para.translation = "[TRANSLATION FAILED]"` — implements D-07 (job continues, never fails)
7. Build and run: `tasks = [translate_one(i, p) for i, p in enumerate(flat)]` then `await asyncio.gather(*tasks)`
8. After gather completes: `dst = _dst_path(src_file, job_dir, target_lang)` then `_write_translated(doc, dst)` — write outside the `async with client:` block is fine; by the time we write, all API calls are done

**Pydantic v2 mutation note** (RESEARCH.md §4): `Paragraph.translation` assignment works in-place (`para.translation = result`) without any `model_config` change. Pydantic v2 models are mutable by default (`frozen` defaults to `False`). The `flat` list holds references to the actual `Paragraph` objects inside `doc`, so mutations are visible in `doc` when serialized.

**Important**: `translate_one` is defined INSIDE `translate` as a closure so it captures `flat`, `client`, `model`, `context_window`, `source_lang`, `target_lang`, `semaphore`, `max_retries` from the enclosing scope without parameter passing.
