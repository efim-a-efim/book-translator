Create `src/book_translator/assembler/html_gen.py` with the following four functions:

**`_inject_class(raw_html: str, css_class: str) -> str`** (private helper):
- Parse `raw_html` with `BeautifulSoup(raw_html, "lxml")`.
- Navigate to the first meaningful element inside the body: `root = soup.body.find()`.
- Add `css_class` to the element's class attribute: `root["class"] = root.get("class", []) + [css_class]`.
- Return `str(root)` (serialized tag only, without the surrounding html/body wrapping that lxml adds).

**`_prefix_ids(html_str: str) -> str`** (private helper):
- Parse with `BeautifulSoup(html_str, "lxml")`.
- For every element where `el.get("id")` is non-empty: rename its `id` to `"bt-orig-" + el["id"]` (per D-10).
- For every `<a>` where `href` starts with `"#"`: update `href` to `"#bt-orig-" + href[1:]` so internal cross-references stay valid.
- Return `str(soup.body.find())` (re-serialized root element only).

**`build_pair_html(para: Paragraph) -> str`**:
- Import `html` from stdlib for escaping.
- If `para.kind in ("image", "table")`: return `para.raw_html` unchanged (D-05 pass-through, no pair wrapper).
- Otherwise (kind is one of `paragraph`, `heading`, `caption`, `footnote`):
  - Apply `_inject_class(para.raw_html, "bt-original")` to get original element with class (per D-02).
  - Apply `_prefix_ids(...)` to that result to prefix any `id` attributes with `bt-orig-` (per D-10).
  - Escape `para.translation or ""` with `html.escape()` to prevent injection from LLM output.
  - Emit: `f'<div class="bt-pair">{original_html}<p class="bt-translation">{escaped_translation}</p></div>'` (D-01, D-03).
- No embedded CSS, no inline styles (D-04).

**`wrap_chapter_xhtml(body_html: str, title: str) -> str`**:
- Escape `title` with `html.escape()`.
- Return EPUB3 XHTML template:
  ```
  <?xml version='1.0' encoding='utf-8'?>
  <!DOCTYPE html>
  <html xmlns="http://www.w3.org/1999/xhtml">
  <head><title>{escaped_title}</title></head>
  <body>{body_html}</body>
  </html>
  ```

Imports needed in `html_gen.py`: `from __future__ import annotations`, `import html as _html`, `from bs4 import BeautifulSoup`, `from book_translator.models.document import Paragraph`.
</action>

**`test_pair_html_paragraph_structure`**: Create `Paragraph(id="p1", text="Hello", raw_html='<p>Hello</p>', translation="Привет", kind="paragraph")`. Call `build_pair_html(para)`. Assert result contains `'<div class="bt-pair">'`, `'class="bt-original"'`, `'<p class="bt-translation">Привет</p>'`.

**`test_pair_html_heading_preserves_tag`**: Create para with `raw_html='<h2>Chapter One</h2>'`, `kind="heading"`, `translation="Глава Первая"`. Assert result contains `'<h2'`, `'class="bt-original"'`, `'<p class="bt-translation">Глава Первая</p>'`. Assert `'<p class="bt-original"'` is NOT in result (original tag preserved, not normalized to p).

**`test_pair_html_image_passthrough`**: Para with `kind="image"`, `raw_html='<img src="cover.png"/>'`, `translation=None`. Assert `build_pair_html(para) == para.raw_html` (exact passthrough, no wrapper).

**`test_pair_html_table_passthrough`**: Para with `kind="table"`, `raw_html='<table><tr><td>cell</td></tr></table>'`, `translation=None`. Assert result equals `para.raw_html`.

**`test_pair_html_translation_html_escaped`**: Para with `kind="paragraph"`, `raw_html='<p>text</p>'`, `translation='<script>alert(1)</script>'`. Assert `'<script>'` does NOT appear in `build_pair_html(para)`. Assert `'&lt;script&gt;'` or equivalent escaped form IS in result.

**`test_pair_html_id_prefixed_on_original`**: Para with `raw_html='<p id="fn1">Footnote</p>'`, `kind="footnote"`, `translation="Сноска"`. Result must contain `'id="bt-orig-fn1"'`. Must NOT contain `'id="fn1"'` outside of the prefix form.

**`test_pair_html_href_anchor_updated`**: Para with `raw_html='<p><a href="#fn1">ref</a> text</p>'`, `kind="paragraph"`, `translation="ref text"`. Assert result contains `'href="#bt-orig-fn1"'` and NOT `'href="#fn1"'`.

**`test_pair_html_none_translation_empty_p`**: Para with `translation=None`, `kind="paragraph"`, `raw_html='<p>text</p>'`. Assert result contains `'<p class="bt-translation"></p>'`.

**`test_wrap_chapter_xhtml_structure`**: Call `wrap_chapter_xhtml("<p>body</p>", "Chapter 1")`. Assert result starts with `"<?xml"`, contains `'xmlns="http://www.w3.org/1999/xhtml"'`, contains `"<title>Chapter 1</title>"`, contains `"<body><p>body</p></body>"`.

**`test_wrap_chapter_xhtml_title_escaped`**: Call `wrap_chapter_xhtml("<p>x</p>", "Title & <More>")`. Assert `"&amp;"` and `"&lt;More&gt;"` appear in the head title.

**`test_pair_html_caption_structure`**: Create `Paragraph(id="c1", text="Caption text", raw_html='<figcaption>Caption text</figcaption>', translation="Подпись.", kind="caption")`. Call `build_pair_html(para)`. Assert result contains `'<div class="bt-pair">'`, `'class="bt-original"'`, `'<figcaption'`, and `'<p class="bt-translation">Подпись.</p>'`.

## Threat Model

| Threat | Category | Component | Disposition | Mitigation |
|--------|----------|-----------|-------------|------------|
| T-04-01 | Injection | `build_pair_html` — `para.translation` inserted into HTML | mitigate | `html.escape()` applied to translation text before insertion; LLM output treated as untrusted |
| T-04-02 | Tampering | `wrap_chapter_xhtml` — chapter title in XHTML head | mitigate | `html.escape()` applied to title string |
| T-04-03 | Tampering | `_inject_class` — class modification via bs4 DOM | accept | bs4 DOM manipulation is safe; raw_html originates from Phase 2 parser, not user input |
