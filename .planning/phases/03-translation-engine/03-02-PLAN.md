**`_is_retryable(exc: BaseException) -> bool`:**
Implements the retry predicate per D-06 and RESEARCH.md §2.

Exception hierarchy fact (verified at runtime): `RateLimitError` inherits from `APIStatusError`. `APIConnectionError` does NOT inherit from `APIStatusError`. Therefore:
- `isinstance(exc, (RateLimitError, APIConnectionError))` → `True` → retryable
- `isinstance(exc, APIStatusError) and exc.status_code >= 500` → `True` → retryable (transient 5xx)
- All other `APIStatusError` (4xx except 429) → `False` → NOT retryable, re-raise immediately

```python
def _is_retryable(exc: BaseException) -> bool:
    if isinstance(exc, (RateLimitError, APIConnectionError)):
        return True
    if isinstance(exc, APIStatusError) and exc.status_code >= 500:
        return True
    return False
```

**`translate_paragraph(client, model, messages, max_retries, semaphore) -> str`:**

Full signature:
```
async def translate_paragraph(
    client: AsyncOpenAI,
    model: str,
    messages: list[dict],
    max_retries: int,
    semaphore: asyncio.Semaphore,
) -> str:
```

The function acquires the semaphore first, then runs the retry loop inside. This means ALL concurrent callers wait at the semaphore boundary before attempting any API call.

Structure:
```
async with semaphore:
    try:
        async for attempt in AsyncRetrying(
            retry=retry_if_exception(_is_retryable),
            stop=stop_after_attempt(max_retries),
            wait=wait_exponential(multiplier=1, min=1, max=60) + wait_random(0, 2),
            before_sleep=before_sleep_log(logger, logging.WARNING),
            reraise=True,
        ):
            with attempt:
                response = await client.chat.completions.create(
                    model=model,
                    messages=messages,
                    temperature=0.3,
                )
                content = response.choices[0].message.content
                if not content or not content.strip():
                    return "[TRANSLATION FAILED]"
                return content.strip()
    except (RateLimitError, APIConnectionError) as exc:
        logger.warning("Translation failed after %d retries: %s", max_retries, exc)
        return "[TRANSLATION FAILED]"
    except APIStatusError as exc:
        if exc.status_code >= 500:
            logger.warning("Translation failed after %d retries (5xx %d): %s", max_retries, exc.status_code, exc)
            return "[TRANSLATION FAILED]"
        raise  # Non-retryable 4xx (401, 403, 400, etc.) — caller handles
```

**Exception ordering is critical**: `except (RateLimitError, APIConnectionError)` must come BEFORE `except APIStatusError` in source order. Python checks `except` clauses top-to-bottom; since `RateLimitError` is a subclass of `APIStatusError`, the first matching clause wins. If `APIStatusError` appeared first, it would catch `RateLimitError` and the `status_code >= 500` check would incorrectly classify 429 (< 500) as non-retryable in the exhaustion path.

**`wait_exponential(multiplier=1, min=1, max=60) + wait_random(0, 2)`**: This adds 0–2 seconds of random jitter on top of the exponential base delay. This prevents thundering-herd when many concurrent paragraphs hit 429 simultaneously and all retry at the same time (D-06 jitter requirement).

**`reraise=True`**: After all retries are exhausted, tenacity re-raises the ORIGINAL exception (not a `RetryError` wrapper). This allows the `except` clauses to catch the original `RateLimitError`/`APIStatusError` by type.
</task>

async def test_semaphore_caps_peak_concurrency() -> None:
    active = 0
    peak = 0

    async def counting_create(**kwargs):
        nonlocal active, peak
        active += 1
        peak = max(peak, active)
        await asyncio.sleep(0.01)
        active -= 1
        resp = MagicMock()
        resp.choices = [MagicMock()]
        resp.choices[0].message.content = "T"
        return resp

    mock_client = MagicMock()
    mock_client.chat.completions.create = counting_create
    sem = asyncio.Semaphore(3)
    msgs = [{"role": "user", "content": "x"}]
    tasks = [translate_paragraph(mock_client, "gpt-4o", msgs, 5, sem) for _ in range(10)]
    results = await asyncio.gather(*tasks)
    assert peak <= 3
    assert all(r == "T" for r in results)
