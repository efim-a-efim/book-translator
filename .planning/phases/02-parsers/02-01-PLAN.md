# Plan 02-01: Foundation — Kind Extension, Parser Protocol, Dependency

**Phase:** 2 — Parsers
**Wave:** 1
**Status:** Complete

## Objective
Prepare the codebase for the parsers phase: extend `Paragraph.kind` with `"image"` and `"table"`, add the `markdown` library dependency, and create the `parsers` package with its `Parser` Protocol and `ParseError` exception.

## Requirements
REQ-1, REQ-2, REQ-3, REQ-7

## Context
Per D-05 and D-16, `Paragraph.kind` must be extended before any parser can emit `"image"` or `"table"` paragraphs — and existing `test_models.py` tests must still pass. Per D-13 and D-15, all three parsers share a `Parser` Protocol and raise `ParseError` on failure. Per D-10, `markdown` is not yet in `pyproject.toml` and must be added before the Markdown parser can be built.

### Task 02-01.3: Create `parsers/__init__.py` — Protocol + ParseError
**Type:** implementation
**Files modified:**
- `src/book_translator/parsers/__init__.py` — new file: `ParseError`, `Parser` Protocol, `__all__`

**Steps:**
1. Create `src/book_translator/parsers/__init__.py`.
2. File structure (in order):
   - `from __future__ import annotations` header (per project convention)
   - `from pathlib import Path`
   - `from typing import Protocol`
   - `from book_translator.models.document import BookDocument`
   - Blank line
   - `class ParseError(ValueError):` — docstring: `"Raised for any unrecoverable parse failure."`; body is just `pass` (message passed via `super().__init__()`)
   - Blank line
   - `class Parser(Protocol):` — docstring: `"Structural protocol — any object with parse(Path) -> BookDocument satisfies it."`; single method: `def parse(self, path: Path) -> BookDocument: ...`
   - Blank line
   - `__all__ = ["ParseError", "Parser"]`

**Verification:** `python -c "from book_translator.parsers import Parser, ParseError; print('OK')"` exits 0.
