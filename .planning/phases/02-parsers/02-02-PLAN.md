# Plan 02-02: EPUB Parser

**Phase:** 2 — Parsers
**Wave:** 2
**Status:** Not Started

## Objective
Implement `EpubParser` in `src/book_translator/parsers/epub.py` — DRM detection, ZIP path traversal guard, spine-order chapter iteration with EpubNav filtering, and a recursive HTML block extractor (`_extract_blocks`) that correctly handles nested `<div>` containers, leaf block tags, and structural elements (`<img>`, `<table>`). The `_extract_blocks` helper is also imported by `md.py` in plan 02-03.

## Requirements
REQ-1, REQ-7

## Context
Per D-17, any EPUB containing `META-INF/encryption.xml` must be rejected with `ParseError` before content extraction begins. Per D-18, ZIP entry paths with `..` components or a leading `/` are also rejected. Per D-01–D-06, block elements are extracted via a recursive descent walk — `<div>` containers are descended into, LEAF_BLOCK_TAGS are collected as paragraphs, and `<img>`/`<table>` are exempt from the D-04 empty-text skip. The critical research finding is that `EpubNav` is a subclass of `EpubHtml`, so the spine filter must be `isinstance(item, epub.EpubHtml) and not isinstance(item, epub.EpubNav)`.

### Task 02-02.1: EPUB security layer — DRM detection + ZIP path traversal guard
**Type:** implementation
**Files modified:**
- `src/book_translator/parsers/epub.py` — new file; define `_check_drm` and `_check_zip_traversal` helpers

**Steps:**
1. Create `src/book_translator/parsers/epub.py` with `from __future__ import annotations` header.
2. Imports: `zipfile`, `pathlib.Path`, `ebooklib.epub`, `bs4.BeautifulSoup`, `book_translator.parsers.ParseError`, and `book_translator.models.document` (Paragraph, Chapter, BookDocument).
3. Implement `_check_drm(path: Path) -> None`:
   - Open `zipfile.ZipFile(path)` in a `with` block.
   - If `"META-INF/encryption.xml"` is present in `zf.namelist()`, raise `ParseError("DRM-protected EPUB — cannot parse")`.
4. Implement `_check_zip_traversal(path: Path) -> None`:
   - Open `zipfile.ZipFile(path)` in a `with` block.
   - For each `name` in `zf.namelist()`: if `".."` in `name.split("/")` OR `name.startswith("/")`, raise `ParseError(f"Unsafe EPUB ZIP entry: {name!r}")`.
   - Note: `name.split("/")` correctly splits POSIX-style ZIP paths; no `os.path` needed.

**Verification:** `python -c "from book_translator.parsers.epub import _check_drm, _check_zip_traversal; print('OK')"` exits 0.

### Task 02-02.2: `_extract_blocks()` — recursive descent HTML block walker
**Type:** implementation
**Files modified:**
- `src/book_translator/parsers/epub.py` — add `LEAF_BLOCK_TAGS`, `TAG_TO_KIND`, `_walk`, `_extract_blocks`

**Steps:**
1. Define module-level constants after imports:
   - `LEAF_BLOCK_TAGS: frozenset[str] = frozenset({"p", "h1", "h2", "h3", "h4", "h5", "h6", "li", "blockquote", "img", "table"})`
   - `TAG_TO_KIND: dict[str, str]` — map: `"h1"` through `"h6"` → `"heading"`, `"blockquote"` → `"caption"`, `"img"` → `"image"`, `"table"` → `"table"`, all others (`"p"`, `"li"`, `"div"`) → `"paragraph"`

2. Define `_walk(node, chapter_id: str, acc: list[Paragraph], counter: list[int]) -> None`:
   - Iterate `node.children`; skip any child without a `.name` attribute or where `.name is None` (these are NavigableString / Comment nodes).
   - **Leaf block** (`child.name in LEAF_BLOCK_TAGS`): compute `text = child.get_text(strip=True)`. Skip if `not text` UNLESS `child.name` is `"img"` or `"table"` (D-04 / D-06 exemption). Append `Paragraph(id=f"{chapter_id}:{counter[0]}", text=text, raw_html=str(child), kind=TAG_TO_KIND.get(child.name, "paragraph"))`. Increment `counter[0]`.
   - **Container div** (`child.name == "div"`): check whether any direct child has `.name in LEAF_BLOCK_TAGS | {"div"}`. If yes → recurse `_walk(child, chapter_id, acc, counter)` (do NOT collect the div itself). If no block children → treat as leaf: apply D-04 skip, collect with `kind="paragraph"`.
   - **Everything else** (e.g. `<section>`, `<article>`, `<nav>`, `<header>`, inline tags): recurse `_walk(child, chapter_id, acc, counter)`.

3. Define `_extract_blocks(html: str, chapter_id: str) -> list[Paragraph]`:
   - `soup = BeautifulSoup(html, "lxml")`
   - `root = soup.find("body") or soup`
   - `paragraphs: list[Paragraph] = []`; `counter = [0]`; call `_walk(root, chapter_id, paragraphs, counter)`; return `paragraphs`.

**Verification:** Inline smoke test: `python -c "from book_translator.parsers.epub import _extract_blocks; r = _extract_blocks('<body><p>hi</p></body>', 'c'); assert r[0].text=='hi' and r[0].id=='c:0' and r[0].kind=='paragraph'; print('OK')"` exits 0.

### Task 02-02.3: `EpubParser` — spine iteration + chapter assembly
**Type:** implementation
**Files modified:**
- `src/book_translator/parsers/epub.py` — add `EpubParser` class and `__all__`

**Steps:**
1. Define `class EpubParser:` (structural match to `Parser` Protocol — no explicit `Protocol` base needed).
2. Implement `def parse(self, path: Path) -> BookDocument:`:
   - Call `_check_drm(path)` then `_check_zip_traversal(path)` first (fail fast before ebooklib touches the file).
   - Open EPUB: `book = epub.read_epub(str(path), options={"ignore_ncx": True})`.
   - Extract metadata: `title = book.title or ""`. For author: `dc_creator = book.get_metadata("DC", "creator")`, `author = dc_creator[0][0] if dc_creator else ""`.
   - Iterate spine: `for spine_id, _ in book.spine:` → `item = book.get_item_with_id(spine_id)`.
   - Filter: skip if `item is None`; skip if NOT `(isinstance(item, epub.EpubHtml) and not isinstance(item, epub.EpubNav))`. (Critical — `EpubNav` is a subclass of `EpubHtml`; the `not isinstance` guard is mandatory.)
   - For each valid item: `chapter_id = item.get_id()`, `html = item.content.decode("utf-8", errors="replace")`, `paragraphs = _extract_blocks(html, chapter_id)`.
   - Skip items where `not paragraphs` (silent skip — covers cover images and empty pages per Agent's Discretion).
   - `Chapter.title` is set to `""` for EPUB chapters — headings are preserved as `kind="heading"` paragraphs and carry the display title; EPUB NCX/OPF spine labels are out of scope for Phase 2.
   - Append `Chapter(id=chapter_id, title="", paragraphs=paragraphs)`.
   - Return `BookDocument(title=title, author=author, chapters=chapters)`.
3. At module bottom: `__all__ = ["EpubParser", "_extract_blocks", "ParseError"]`
   (Exporting `_extract_blocks` for `md.py` to import; underscore prefix signals internal use.)

**Verification:** `python -c "from book_translator.parsers.epub import EpubParser; print('OK')"` exits 0. Full integration is verified by tests in plan 02-03.

## Threat Model
- **ZIP path traversal (T-02-01):** `_check_zip_traversal` rejects entry paths containing `".."` or starting with `"/"` per D-18. Defense-in-depth — ebooklib internally handles most traversal cases, but an explicit validation layer is required.
- **DRM-encrypted EPUB (T-02-02):** `_check_drm` checks `META-INF/encryption.xml` and raises `ParseError` before any content is read per D-17. Disposition: mitigate.
- **Malformed EPUB HTML (T-02-03):** BeautifulSoup with the `lxml` backend handles malformed EPUB XHTML gracefully without raising. No `eval`/`exec` code paths exist. Disposition: accept (local CLI tool).
- **Memory exhaustion from very large EPUB (T-02-04):** No streaming — entire EPUB is loaded into memory. Disposition: accept (local CLI; not a server).
